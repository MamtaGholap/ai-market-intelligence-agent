Process Flow
01

User Input

User launches the tool and enters a stock ticker symbol (e.g. AAPL, NVDA, TSLA). Input is validated and converted to uppercase automatically.

input() .upper() CLI Interface
02

Live Stock Data Fetch

Python calls the Alpha Vantage financial API in real time. Returns current price, daily change percentage, volume, and last trading day. Data is parsed from JSON response.

Alpha Vantage API requests.get() JSON parsing Live Market Data
03

AI Prompt Engineering

Stock data is structured into a professional financial analyst prompt using Python f-strings. The prompt instructs the AI to produce a market summary, price movement analysis, and risk/opportunity assessment.

f-strings Prompt Design Context Injection
04

Google Gemini AI Analysis

Structured prompt is sent to Google Gemini AI via the official API. The model generates analyst-grade financial commentary including market context, risk factors, and investment opportunities. Error handling manages API failures gracefully.

Google Gemini API google-genai try/except AI Generation
05

Report Generation & Auto-Save

Combined stock data and AI analysis are formatted into a structured professional report. File is automatically named using the stock symbol and saved to disk using Python file handling.

File I/O open() with statement Auto-naming AAPL_report.txt
06

Output — Analyst-Grade Report

User receives a complete market intelligence report: live price data + AI-generated analysis covering market sentiment, price movement indicators, investment risks and opportunities. Report persists on disk for future reference.

Professional Report Persistent Storage Ready for Presentation
